// Data & Privacy Consent

Before using simper~decide, please review how your data is handled:

What is stored (locally on your device only)
🔑 API Key — your Claude API key, stored in browser localStorage
👤 Profile — name, age, location, background, photo (optional)
💬 Sessions — up to 5 chat sessions saved locally
What is NOT stored
✕ No data is sent to any server except Anthropic's Claude API
✕ No tracking, no analytics, no third-party services
✕ Camera photos are processed in-memory only — not saved to disk

You can view or delete all stored data at any time via Settings → My Data.
